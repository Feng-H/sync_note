# 音视频文字同步记录

一个简洁高效的音频记录应用，支持边播放录音边输入文字，并实现时间戳同步功能。

## 功能特性

- **音频上传与播放**：支持上传各种格式的音频文件，提供播放/暂停控制
- **实时文字记录**：在播放音频的同时，使用 Markdown 编辑器记录内容
- **时间戳同步**：自动记录每段文字输入时的音频播放时间点
- **智能跳转**：点击预览区的任意文字，音频自动跳转到该文字记录时的时间点（提前5秒）
- **快捷键控制**：
  - 点击播放器区域后，使用 ← 键快退10秒
  - 点击播放器区域后，使用 → 键快进10秒
- **Markdown 支持**：支持 Markdown 格式，实时预览渲染效果

## 使用场景

- 会议记录
- 采访整理
- 课程笔记
- 播客内容记录
- 音频内容转写

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 使用说明

1. 点击"上传录音文件"按钮，选择音频文件
2. 点击"播放"按钮开始播放音频
3. 在左侧编辑器中输入文字内容（支持 Markdown 语法）
4. 右侧预览区会实时显示渲染后的内容
5. 点击预览区的任意文字，音频会跳转到该文字记录时的时间点（提前5秒）
6. 点击播放器区域后，可使用左右方向键快退/快进10秒
